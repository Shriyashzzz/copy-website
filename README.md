# Landing Page Project

## Overview

This project is a **basic landing page website** built using **HTML and CSS**. The goal of this project was to practice structuring a webpage and styling it using CSS, with a focus on **Flexbox layout techniques**.

The page is divided into **four main sections and a footer**, following a provided design reference. The project was completed by building the HTML structure first and then applying CSS styling.

## Features

* Hero section with main headline and call-to-action
* Multiple content sections
* Flexbox layout for alignment and spacing
* Simple and clean design
* Footer section

## Project Structure

```
copy-website/
│
├── index.html

└── README.md
```

## Technologies Used

* HTML5
* CSS3
* Flexbox

## Design Notes

The webpage was built based on a provided design that included:

* A **full layout mockup**
* **Color palette and typography guide**

The page was implemented section by section:

1. Hero Section
2. Information Section
3. Quote/Testimonial Section
4. Call-to-Action Section
5. Footer

The project was designed for **desktop viewing only** and is **not responsive**.
